# Install FoundryOS

This guide assumes zero prior experience. If you've never used GitHub or an AI coding assistant before, you're in the right place.

FoundryOS isn't software you run — it's a folder of markdown files that an AI assistant reads. "Installing" it just means getting that folder somewhere your assistant can see it.

## Step 1: Download the ZIP

1. Go to the FoundryOS GitHub repository.
2. Click the green **Code** button.
3. Click **Download ZIP**.
4. Save it somewhere you'll remember, like your Desktop or Documents folder.

## Step 2: Extract the folder

- **Windows:** Right-click the downloaded `.zip` file → **Extract All** → choose a location → **Extract**.
- **Mac:** Double-click the downloaded `.zip` file. It unzips automatically into the same folder.

You should now have a folder called `FoundryOS` containing subfolders like `modules/`, `skills/`, `agents/`, and `commands/`.

## Step 3: Choose your environment

Pick whichever AI assistant you already use:

| If you use… | Go to… |
|---|---|
| Claude (claude.ai, Claude Projects, or Claude Code) | [`docs/CLAUDE_SETUP.md`](docs/CLAUDE_SETUP.md) |
| ChatGPT (Projects or Custom GPTs) | [`docs/CHATGPT_SETUP.md`](docs/CHATGPT_SETUP.md) |
| Cursor | [`docs/CURSOR_SETUP.md`](docs/CURSOR_SETUP.md) |
| Windsurf | [`docs/WINDSURF_SETUP.md`](docs/WINDSURF_SETUP.md) |

Don't have any of these yet? Claude is the easiest starting point — see [`docs/CLAUDE_SETUP.md`](docs/CLAUDE_SETUP.md).

## Step 4: Follow the setup guide

Each setup guide walks you through uploading or opening the folder, then gives you a first prompt to try. Once that's done, you're up and running — see [`GETTING_STARTED.md`](GETTING_STARTED.md) for what to do next.
