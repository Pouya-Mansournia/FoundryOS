# 47 Naming Skill

## What this skill does
Generates and screens naming candidates for a company, product, or feature against naming criteria, trademark conflicts, and domain availability.

## Source Modules
- 138_Naming_OS
- 139_Naming_Convention_OS
- 140_Trademark_Screening_OS

## Output
- Naming Shortlist
- Naming Criteria
- Naming Convention Guide
- Trademark Screening Summary
- Domain Availability Check
