# 53 Voice & Tone Skill

## What this skill does
Defines the brand's voice and the tone spectrum it flexes across by context and channel, so the same brand sounds consistent under different conditions.

- 157_Tone_Spectrum_OS
- 158_Voice_Guidelines_OS

- Channel Tone Matrix
- Words-to-Use / Words-to-Avoid List
- Voice Consistency Checklist
