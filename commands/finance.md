# Command
/finance

## Purpose
Models unit economics, P&L, and scenario analysis — the financial core without the investor narrative (`/fundraising` builds on top of this).

## Activated Agents
- CFO-Agent

## Activated Skills
- `07-finance-skill` — unit economics, P&L, scenarios

## Workflows
- `06-fundraising-workflow`
- `10-strategic-planning-workflow`

## Output
- Unit Economics
- P&L
- Cash Flow, Burn Rate & Runway
- Scenario Analysis (best/base/worst case)

## Example
`/finance Build an 18-month cash flow model under best/base/worst-case assumptions.`
