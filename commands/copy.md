# Command
/copy

## Purpose
Activates CBO-Agent to write or rewrite copy — landing pages, emails, in-product strings, ads — against the voice rules already established, instead of producing generic marketing language that has to be re-edited into the brand's actual voice afterward.

## Activated Agents
- CBO-Agent

## Activated Skills
- `52-copywriting-skill` — headline/body/CTA writing, compression, rhythm
- `53-voice-tone-skill` *(input — supplies the tone-by-channel rules copy must follow)*

## Workflows
- `07-go-to-market-workflow`

## Output
- Copy Draft (channel-specific: landing page, email, in-product, ad)
- Voice Compliance Notes (where the draft follows or deviates from `memory/voice-memory.md`, and why)

## Example
`/copy Write the email announcing our v2 launch — three variants, one direct, one playful, one technical — and flag which fits our actual brand voice.`
