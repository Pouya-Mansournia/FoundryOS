# Command
/website

## Purpose
Activates CBO-Agent to design the website as a synthesis artifact — UI system, copy, and design tokens applied to one customer-facing surface — rather than treating layout, words, and visuals as three separate handoffs that drift out of sync with each other.

## Activated Agents
- CBO-Agent

## Activated Skills
- `56-ui-system-skill` — page-level component application
- `44-design-system-skill` — tokens the page is built against
- `52-copywriting-skill` — headline, body, and CTA copy

## Workflows
- `07-go-to-market-workflow`
- `02-saas-product-workflow`

## Output
- Website Structure (sitemap, page-level wireframe intent)
- Page Copy (headline, subhead, body, CTAs)
- Component Spec (built against the existing design system)

## Example
`/website Draft the homepage for our SaaS product launch — hero, three feature sections, pricing teaser, and footer, using our existing design tokens.`
