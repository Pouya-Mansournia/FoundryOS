# Command
/fundraising

## Purpose
Generates the artifacts needed to be investor-ready — the financial model, product scorecard, and pitch narrative a fundraising conversation runs on.

## Activated Agents
- CEO-Agent
- CFO-Agent
- CPO-Agent
- CRO-Agent

## Activated Skills
- `38-fundraising-investor-skill` — pitch narrative, data room, investor comms
- `07-finance-skill` — financial model, scenarios
- `27-product-scorecard-skill` — PMF/retention scoring
- `08-gtm-skill` — traction narrative

## Workflows
- `06-fundraising-workflow`

## Output
- Pitch Narrative
- Financial Model & Scenario Analysis
- Data Room Checklist
- Product Scorecard
- Investor Update Template

## Example
`/fundraising Get us investor-ready for a raise in the next few months.`
