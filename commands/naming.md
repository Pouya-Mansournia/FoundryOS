# Command
/naming

## Purpose
Activates CBO-Agent to generate and screen name candidates — for a company, a product line, or a single feature — against trademark risk, domain availability, pronounceability, and fit with the brand archetype already in `memory/brand-memory.md`.

## Activated Agents
- CBO-Agent

## Activated Skills
- `47-naming-skill` — name generation, screening, shortlist
- `42-brand-strategy-skill` *(upstream — archetype must exist before names are screened against it)*

## Workflows
- `01-new-product-workflow`
- `05-robotics-product-workflow`

## Output
- Naming Shortlist (5-10 candidates with rationale)
- Screening Notes (trademark, domain, pronounceability flags)
- Recommended Name

## Example
`/naming We need a name for our new AI copilot feature — should feel precise and trustworthy, not playful.`
