# Command
/cfo

## Purpose
Activates CFO-Agent for unit economics, financial planning, scenario analysis, and the fundraising narrative built on top of the numbers.

## Activated Agents
- CFO-Agent

## Activated Skills
- `07-finance-skill` — unit economics, pricing inputs, financial scenarios
- `38-fundraising-investor-skill` — fundraising narrative, data room, investor comms

## Workflows
- `06-fundraising-workflow`
- `10-strategic-planning-workflow`

## Output
- Pricing Inputs
- Financial Model (P&L, Unit Economics)
- Cash Flow / Burn Rate / Runway
- Scenario Analysis (best/base/worst case)

## Example
`/cfo Build an 18-month cash flow model with best/base/worst-case scenarios for our seed round.`
